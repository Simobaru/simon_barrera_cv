## Idea 1: "The Blueprint Architect" (Ingeniería de Precisión)

*   **Design Movement**: Brutalismo Técnico / Swiss Style Moderno.
*   **Core Principles**:
    1.  **Estructura Expuesta**: Líneas de rejilla visibles, coordenadas, y elementos que recuerdan a planos técnicos (blueprints).
    2.  **Funcionalidad Pura**: Cada elemento tiene un propósito claro, sin adornos innecesarios.
    3.  **Precisión Matemática**: Alineaciones perfectas, uso de monoespaciado para datos técnicos.
*   **Color Philosophy**:
    *   Fondo: Azul Oscuro Profundo (#0B1F33) para evocar el espacio de trabajo de ingeniería nocturno o un entorno CAD.
    *   Acentos: Cian Eléctrico (#2FA4FF) para resaltar puntos de interacción y datos clave, simulando láseres o indicadores digitales.
    *   Texto: Blanco puro y gris técnico para máxima legibilidad y contraste.
    *   *Intención*: Transmitir rigor, exactitud y la naturaleza técnica del perfil.
*   **Layout Paradigm**: Asimétrico y modular. Bloques de contenido que parecen paneles de control o fichas técnicas. Uso de líneas divisorias finas (1px) para separar secciones.
*   **Signature Elements**:
    1.  **Grid Lines & Crosshairs**: Líneas de guía sutiles y cruces en las esquinas de los contenedores.
    2.  **Technical Badges**: Etiquetas de habilidades que parecen especificaciones de componentes.
    3.  **Data Visualization**: Gráficos de barras o radiales minimalistas para mostrar niveles de habilidad o estadísticas de proyectos.
*   **Interaction Philosophy**: "Feedback Háptico Visual". Los botones y cards reaccionan instantáneamente con cambios de borde o color, como interruptores físicos o digitales de alta respuesta.
*   **Animation**:
    *   Transiciones tipo "Typewriter" o "Data Reveal" para textos.
    *   Líneas que se dibujan al hacer scroll (SVG path drawing).
    *   Elementos que se ensamblan o deslizan con precisión mecánica (easing: cubic-bezier).
*   **Typography System**:
    *   Títulos: *Space Grotesk* o *Chakra Petch* (Sans-serif con carácter técnico/futurista).
    *   Cuerpo/UI: *JetBrains Mono* o *Roboto Mono* para datos, *Inter* para lectura larga.
    *   *Regla*: Uso extensivo de mayúsculas para encabezados de sección (tracking extendido).
</text>

## Idea 2: "Deep Tech Visionary" (Elegancia Futurista)

*   **Design Movement**: Glassmorphism Oscuro / Cyber-Noir Elegante.
*   **Core Principles**:
    1.  **Profundidad Inmersiva**: Uso de capas, desenfoques (backdrop-filter) y transparencias para crear un sentido de espacio 3D.
    2.  **Luz como Guía**: La iluminación (glows, sombras de colores) dirige la atención del usuario.
    3.  **Fluidez Orgánica**: Formas más suaves y transiciones líquidas que contrastan con la rigidez técnica, sugiriendo adaptabilidad e innovación.
*   **Color Philosophy**:
    *   Fondo: Gradientes muy sutiles de Azul Noche (#0E2A47) a Negro, creando una atmósfera de "laboratorio avanzado".
    *   Acentos: Azul Neón suave y toques de violeta muy oscuro para sombras, dando un aire sofisticado y moderno.
    *   *Intención*: Evocar innovación, futuro y la sofisticación de la IA y la robótica avanzada.
*   **Layout Paradigm**: Flotante y centrado en el contenido. Las secciones son "tarjetas" de vidrio esmerilado que flotan sobre un fondo animado sutilmente.
*   **Signature Elements**:
    1.  **Glass Cards**: Contenedores con bordes translúcidos y fondo semitransparente.
    2.  **Glow Effects**: Resplandores suaves detrás de elementos clave (como el avatar o proyectos destacados).
    3.  **Abstract 3D Background**: Partículas o mallas 3D (Three.js) que reaccionan suavemente al mouse en el fondo.
*   **Interaction Philosophy**: "Levitación y Profundidad". Al hacer hover, los elementos se elevan (transform: translateY) y la luz/sombra cambia, sugiriendo tridimensionalidad.
*   **Animation**:
    *   Fade-in suave y escalado lento para la entrada de elementos.
    *   Parallax sutil entre el fondo y las tarjetas de contenido.
    *   Efectos de "resplandor" que recorren los bordes de los botones.
*   **Typography System**:
    *   Títulos: *Playfair Display* (Serif moderna) para un toque de autoridad y elegancia clásica contrastando con el entorno tech.
    *   Cuerpo: *Manrope* o *Plus Jakarta Sans* (Geométricas modernas) para una legibilidad limpia y contemporánea.
</text>

## Idea 3: "The Strategic Leader" (Editorial Tech)

*   **Design Movement**: Minimalismo Editorial / Swiss International Style.
*   **Core Principles**:
    1.  **Contenido es Rey**: Tipografía grande, márgenes generosos y fotografía de alta calidad (o renders abstractos).
    2.  **Jerarquía Clara**: Uso dramático del tamaño de fuente para guiar la lectura.
    3.  **Sofisticación Sobria**: Menos es más. Eliminar todo ruido visual para centrarse en el mensaje de liderazgo y logros.
*   **Color Philosophy**:
    *   Fondo: Blanco (#FFFFFF) o Gris Perla muy claro (#F8F9FA) para la mayoría de secciones, con una sección Hero en Azul Oscuro (#0B1F33) para impacto inicial.
    *   Texto: Azul Marino muy oscuro (#05101A) para máximo contraste y seriedad.
    *   Acentos: Azul Real (#2FA4FF) solo para enlaces y llamadas a la acción (CTAs).
    *   *Intención*: Transmitir claridad, confianza, madurez profesional y transparencia.
*   **Layout Paradigm**: Grid editorial fuerte. Texto a una o dos columnas con mucho aire alrededor. Imágenes grandes que rompen la rejilla ocasionalmente.
*   **Signature Elements**:
    1.  **Big Typography**: Encabezados de sección enormes que funcionan como elementos gráficos.
    2.  **Lineas de Separación Gruesas**: Uso de líneas horizontales negras o azules para separar temas.
    3.  **Iconografía Minimalista**: Iconos de línea fina (stroke) sin relleno.
*   **Interaction Philosophy**: "Micro-interacciones Precisas". Animaciones muy rápidas y sutiles. El foco está en la legibilidad y la navegación fluida.
*   **Animation**:
    *   Scroll reveal simple (aparecer desde abajo).
    *   Imágenes que se revelan con una máscara (clip-path).
    *   Sticky headers o elementos laterales que acompañan la lectura.
*   **Typography System**:
    *   Títulos: *Libre Baskerville* o *Merriweather* (Serif robusta) para transmitir tradición y solidez.
    *   Cuerpo: *Inter* o *Helvetica Now* para neutralidad y perfección suiza.
</text>
